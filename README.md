# Stack Overflow Developer Survey Data Analysis

## Project Overview

This project analyzes the Stack Overflow Developer Survey dataset using Python. The objective is to clean, explore, visualize, and extract meaningful insights about the global developer community.

The analysis covers demographics, professional experience, developer roles, and country-wise participation using exploratory data analysis (EDA) techniques and interactive visualizations.

--- 

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly Express

---

## Project Workflow

- Imported and explored the dataset
- Created a working copy with selected features
- Inspected and handled missing values
- Cleaned categorical data (e.g., gender, developer type)
- Conducted demographic and professional analysis
- Built visualizations for:
- Top 10 countries by number of respondents
- Gender distribution
- Age distribution
- Developer type distribution
- Professional coding experience

## Key Insights
- The developer community is globally diverse but concentrated in a few countries.
- Gender representation is skewed, with men forming the majority.
- Most developers are in early to mid-career stages.
- Full-stack and back-end roles dominate the profession.
- Coding experience varies widely, reflecting both fresh entrants and experienced professionals.
